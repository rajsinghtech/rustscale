## 1. WIRE tshttpproxy into outbound connections

### Goal

Route rustscale's outbound control-plane (controlhttp) and DERP TCP/TLS connections through the system HTTP CONNECT proxy when `http_proxy`/`https_proxy` env vars are set. The `rustscale_tshttpproxy` crate already has `proxy_from_environment()` — make three call sites use it.

### Files to modify

**A. `crates/controlclient/src/controlhttp.rs`** — two functions need proxying:

Current `dial_control()` at line 252:
```rust
pub async fn dial_control(
    url: &str,
    machine_key: &MachinePrivate,
    control_key: &MachinePublic,
    version: ProtocolVersion,
) -> Result<NoiseStream<AnyStream>, DialError> {
    let parsed = parse_control_url(url);
    if parsed.is_plain {
        let tcp = tokio::net::TcpStream::connect((parsed.host.as_str(), parsed.port))
            .await
            .map_err(DialError::Io)?;
        let stream = AnyStream::Plain(tcp);
        // ...
    } else {
        let insecure = is_insecure_host(&parsed.host);
        let tls_stream = tls_connect(&parsed.host, parsed.port, insecure).await?;
        let stream = AnyStream::Tls(Box::new(tls_stream));
        // ...
    }
}
```

Current `fetch_server_pub_key()` at line 285 — also does raw `TcpStream::connect` (lines 293, 313).

Current `tls_connect()` at line 387 — does raw TCP then TLS.

**Strategy** (simpler than Go's feature-hook pattern; Rust doesn't need the hook indirection):

1. Add a `fn proxy_from_env_for(url: &str) -> Option<Url>` helper that calls `rustscale_tshttpproxy::proxy_from_environment()` on the target URL (scheme-appropriate: for `https://` control URLs, use the HTTPS proxy; for `wss://` DERP URLs in plain TCP mode, use HTTPS proxy too).

2. In `dial_control()`: before connecting, check the proxy. If present, perform HTTP CONNECT:
   - Connect `TcpStream` to the proxy address (host:port from proxy URL)
   - Send `CONNECT {target_host}:{target_port} HTTP/1.1\r\nHost: {target_host}:{target_port}\r\n\r\n`
   - Read the HTTP status line: must start with `HTTP/1.1 200` — if not, return `DialError::Proxy(format!("..."))`
   - Use the resulting `TcpStream` (now tunneled through the proxy) as the base stream for the TLS or plain upgrade.
   - Also handle Proxy-Authorization: if the proxy URL has userinfo (`username:password@host:port`), add `Proxy-Authorization: Basic base64(user:pass)` header to the CONNECT request (Go's `tshttpproxy.GetAuthHeader` does this).

3. In `fetch_server_pub_key()`: same proxy logic for the `/key` GET request — wrap the raw TCP in a proxy connect.

4. In `tls_connect()`: same — check proxy before dialing, CONNECT through proxy if present.

**Strategy** for DERP (matches Go's `derphttp.dialNode()` / `dialNodeUsingProxy()`):
- Go checks proxy BEFORE connecting. If present, calls `dialNodeUsingProxy()` which does raw CONNECT — NOT the HTTP upgrade path. For DERP over proxy, downgrade from "connect with upgrade" to simple TLS connect through the tunnel.
- In `connect_insecure()` / `connect_with_upgrade_dial_insecure()`: call `proxy_from_environment(&url)` for the target DERP URL with scheme `https://` + host:port. If proxy returns `Some(url)`:
  - Connect `TcpStream` to `proxy_url.host:proxy_url.port`
  - Send HTTP CONNECT to `host:port`
  - If use_tls: wrap the tunneled TCP in TLS (with the original tls_host for SNI). **Skip the HTTP upgrade** — just do TLS + DERP handshake directly, matching Go's behavior.
- Go's actual CONNECT code in `derphttp.dialNodeUsingProxy()`: dials proxy, sends `CONNECT n.Addr() HTTP/1.1\r\nHost: n.Addr()\r\n[Proxy-Authorization: ...]\r\n`, reads `HTTP/1.1 200`. Use the same pattern.

**C. `crates/tsnet/src/localapi.rs`** — Not a priority for this phase. The localapi serves on localhost and doesn't make outbound connections.

### Key facts from Go

- `tshttpproxy.ProxyFromEnvironment(req.url)` returns `*url.URL` or nil — wrap with `use std::str::FromStr; Url::from_str(proxy_str)`.
- Go's controlhttp sets `tr.Proxy` on `http.Transport` (automatic CONNECT for RoundTrip). Rust doesn't use `http.Transport` — must do CONNECT manually.
- Go's DERP `dialNode()` uses separate proxy check per-region, calling `proxyFromEnv(proxyReq)` where `proxyReq.URL` is `&url.URL{Scheme: "https", Host: n.Addr()}`.
- `no_proxy` EXEMPTIONS are handled inside `proxy_from_environment()` already (the crate was built for this). So `tshttpproxy::proxy_from_environment(url::Url::parse("https://control.tailscale.com:443")?)` will return `None` if the host matches `no_proxy`.
- `set_self_proxy` prevents proxy loops: call `tshttpproxy_set_self_proxy(&["127.0.0.1", "::1"])` in the server bootstrap (tsnet lifecycle).

### Dependencies to add

- `rustscale-tshttpproxy` to `controlclient/Cargo.toml` and `derp/Cargo.toml` workspace deps
- No new HTTP client crate needed — just `tokio::net::TcpStream` and manual HTTP/1.1 CONNECT parsing

**4. Background upload loop** (tokio task, mirrors Go's `uploading()` goroutine):

```rust
async fn upload_loop(config: Config, buffer: Arc<Mutex<VecDeque<LogEntry>>>,
    flush_trigger: tokio::sync::watch::Receiver<bool>,
    shutdown: tokio::sync::watch::Receiver<bool>) 
{
    // Use reqwest::Client (or hyper) for HTTP POST
    let client = reqwest::Client::builder()
        .user_agent("")  // Go removes User-Agent
        .build()
        .unwrap();

    loop {
        // Wait for either:
        // - flush_trigger fires (after write or explicit flush)
        // - flush_delay elapses (2-second batching, Go's FlushDelayFn default)
        // - shutdown fires
        
        let entries = drain_buffer(&buffer);
        if entries.is_empty() {
            if *shutdown.borrow() { return; }
            // Wait for new data or shutdown
            tokio::select! {
                _ = flush_trigger.changed() => continue,
                _ = shutdown.changed() => if *shutdown.borrow() { return; },
            }
            continue;
        }

        let body = serialize_entries(&entries);
        let body = if config.compress_logs && body.len() > 256 {
            // Try zstd compression; only use if savings > 64 bytes
            // Set origlen = body.len(), then body = compressed
        } else {
            body
        };

        // Upload with retry
        let mut last_error = String::new();
        let mut failures: u32 = 0;
        let mut first_failure: Option<Instant> = None;
        
        loop {
            match upload_single(&client, &config, &body, origlen).await {
                Ok(()) => {
                    if failures > 0 {
                        // log recovery
                    }
                    break;
                }
                Err((retry_after, err_str)) => {
                    failures += 1;
                    first_failure.get_or_insert_with(Instant::now);
                    
                    // Check if internet is up (always assume up for now)
                    
                    if last_error != err_str {
                        log::warn!("logtail: upload: {err_str}");
                        last_error = err_str;
                    }
                    
                    let delay = if retry_after > 0 {
                        std::cmp::min(retry_after, Duration::from_secs(300))
                    } else {
                        // Random 30-60s
                        Duration::from_secs(30 + (rand::random::<u64>() % 31))
                    };
                    tokio::time::sleep(delay).await;
                }
            }
            
            if *shutdown.borrow() { return; }
        }
    }
}
```

**5. `upload_single()` function:**
```rust
async fn upload_single(client: &reqwest::Client, config: &Config, body: &[u8], origlen: Option<usize>) 
    -> Result<(), (Duration, String)>
{
    let url = format!("{}/c/{}/{}",
        config.base_url(), config.collection, config.private_id);
    
    let mut req = client.post(&url)
        .header("User-Agent", "")  // suppress User-Agent
        .timeout(Duration::from_secs(45));
    
    if let Some(orig) = origlen {
        req = req
            .header("Content-Encoding", "zstd")
            .header("Orig-Content-Length", orig.to_string());
    }
    
    let resp = req.body(body.to_vec()).send().await
        .map_err(|e| (Duration::ZERO, e.to_string()))?;
    
    if resp.status() != 200 {
        let status = resp.status();
        let retry_after = resp.headers()
            .get("Retry-After")
            .and_then(|v| v.to_str().ok())
            .and_then(|v| v.parse::<u64>().ok())
            .unwrap_or(0);
        let resp_body = resp.text().await.unwrap_or_default();
        return Err((Duration::from_secs(retry_after),
            format!("upload failed {status}: {resp_body}")));
    }
    
    Ok(())
}
```
