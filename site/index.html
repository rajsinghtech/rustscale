Install rustscale

v0.1.1

A Rust implementation of Tailscale's client stack, with familiar CLI and container workflows for drop-in deployments.

macOS and Linux

Installs verified release binaries and the C library. System service setup is automatic where supported.

curl -fsSL https://rajsinghtech.github.io/rustscale/install.sh | sh

Windows PowerShell

User-scoped by default, with checksum verification and no administrator requirement.

irm https://rajsinghtech.github.io/rustscale/install.ps1 | iex

Homebrew

Installs the rustscale CLI and rustscaled daemon on macOS or Linux.

brew install rajsinghtech/tap/rustscale

Container

Multi-architecture image with Tailscale-compatible environment variables and command aliases.

docker pull ghcr.io/rajsinghtech/rustscale:v0.1.1
Replacing an upstream installation? Download the Unix installer and run it with --tailscale-compatible. The aliases are explicit so rustscale never silently overwrites an installed official client.
