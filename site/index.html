rustscale

A Rust implementation of Tailscale's client stack — direct UDP, DERP relay, and peer-relay in one binary.

curl -fsSL https://rajsinghtech.github.io/rustscale/install.sh | sh
irm https://rajsinghtech.github.io/rustscale/install.ps1 | iex
brew install rajsinghtech/tap/rustscale

Quick start

sudo rustscaled run          # start the daemon
rustscale up                 # connect to a tailnet
rustscale status             # check state
rustscale whois 100.64.0.2   # peer identity

WireGuard data plane

Userspace tunnel via boringtun — no kernel module required.

Direct + DERP + relay

UDP hole-punching, DERP relay fallback, and peer-relay for hard NATs.

C FFI — librustscale

Embed in any language. Static and dynamic libraries, single header.

Faster than tailscaled

p50 ~170µs vs 257µs. Throughput 465–838 vs 384 Mbps.
